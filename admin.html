텔레그램 봇 연동 미연결

신규 상담 신청 시 텔레그램으로 즉시 알림을 받을 수 있습니다.

📱 텔레그램 봇 설정 가이드

  1. 텔레그램에서 @BotFather 검색 후 대화 시작
  2. /newbot 입력 → 봇 이름·아이디 입력 → BOT TOKEN 발급
  3. 새로 만든 봇과 대화 시작 후 아무 메시지나 전송
  4. @userinfobot으로 본인 CHAT ID 확인
  5. 위에 입력 → [저장] → [테스트 전송]으로 정상 작동 확인

👥 추가 관리자 계정 만들기

  1. Supabase 대시보드 접속
  2. 좌측 메뉴 Authentication → Users
  3. 우측 상단 Add user → Create new user 클릭
  4. 이메일, 비밀번호 입력 → Auto Confirm User 체크 → 생성
