📱 텔레그램 봇 설정 가이드

  1. 텔레그램에서 @BotFather 검색 후 대화 시작
  2. /newbot 입력 → 봇 이름·아이디 입력 → BOT TOKEN 발급
  3. 새로 만든 봇과 대화 시작 후 아무 메시지나 전송 (그룹이면 봇을 그룹에 초대)
  4. @userinfobot으로 본인 또는 그룹의 CHAT ID 확인
  5. 위에 입력 → [저장] → [테스트 전송]으로 정상 작동 확인
